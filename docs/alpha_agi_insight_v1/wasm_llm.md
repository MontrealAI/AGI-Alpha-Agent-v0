# Insight WASM LLM Assets

The Insight demo ships a lightweight WebAssembly LLM runtime for offline mode. The assets live under
[`docs/alpha_agi_insight_v1/assets/wasm_llm/`](assets/wasm_llm/) and mirror the demo's build output.

For details on regenerating or refreshing these assets, see the README in the asset directory:
[`assets/wasm_llm/README.md`](assets/wasm_llm/README.md).
