# Insight WASM LLM Assets

This page documents the WebAssembly-based LLM artifacts used by the Insight v1 demo.
The assets are stored under `docs/alpha_agi_insight_v1/wasm_llm/` and are packaged
alongside the static demo files for offline use.

If you need to refresh the runtime, follow the browser asset update instructions in
`alpha_factory_v1/scripts/README.md` and regenerate the demo assets. Keep the
`docs/alpha_agi_insight_v1/wasm_llm/` directory in sync with those updates.
