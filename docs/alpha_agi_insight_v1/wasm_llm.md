# Insight WASM LLM Assets

The Insight demo ships lightweight WASM LLM assets for offline inference. The files
live under `docs/alpha_agi_insight_v1/assets/wasm_llm/` and mirror the assets used
by the Insight browser demo.

For the full asset manifest and update instructions, see
`docs/alpha_agi_insight_v1/assets/wasm_llm/README.md`.
