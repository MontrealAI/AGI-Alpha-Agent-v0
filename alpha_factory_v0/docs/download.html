Alpha‑Factory Pro – Downloads

Build & run your own instance in minutes.

1. Clone the repo

git clone https://github.com/your‑org/alpha‑factory.git
cd alpha‑factory

2. Build the Docker image

Interactive:

./install_alpha_factory_pro.sh

Headless server, no UI:

./install_alpha_factory_pro.sh --no-ui --trace --no-cache
Download installer

3. Run

docker run --rm -p 33000:3000 alphafactory_pro:latest

Open http://localhost:33000/api/logs to verify.
